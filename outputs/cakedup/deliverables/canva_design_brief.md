## Brand Identity

| Element | Value |
|---|---|
| Business name | Cake'd Up |
| Vibe | Elegant, girly, professional — think: Tiffany & Co. meets a bakery |
| Fonts | Heading: **Pacifico** or **Dancing Script** · Body: **Nunito** or **Lato** |

## Color Cheat Sheet

```
Tiffany Blue   #0ABAB5   → headers, buttons, main color
Light Tiffany  #E0F7F6   → backgrounds, card fills
Silver         #B0B8C1   → prices, highlights, accents
White          #FFFFFF   → text on Tiffany Blue backgrounds
Dark Charcoal  #1F2937   → all body text on light backgrounds
```

## Design 1 — Logo

**Size:** Any (design as vector/text-based — no fixed size needed)

**Instructions:**
1. Open Canva → Create a design → Logo
2. Add a text box → type `Cake'd Up`
3. Font: **Pacifico** · Size: 60–80pt · Color: `#0ABAB5`
4. Add a small cake emoji or cake icon beside the text
   - Search Canva elements: "cake" → pick a simple, elegant one
   - Color the icon silver `#B0B8C1`
5. Optional: add a thin circle border in Tiffany Blue around the whole thing
6. **Save as PNG with transparent background** (File → Download → PNG → check "Transparent background")

## Tiffany Aesthetic Tips

- **White ribbon on everything** — Tiffany's signature is the white ribbon on the blue box. Use white ribbon/bow elements wherever you can in Canva
- **Clean, uncluttered layouts** — Tiffany aesthetic is spacious and minimal. Don't crowd the designs
- **Silver over gold** — silver jewelry is Tiffany's signature. Use silver `#B0B8C1` where other brands would use gold
- **Blue + white only** — the power of the Tiffany look is restraint. Two colors, done beautifully, beats five colors every time
